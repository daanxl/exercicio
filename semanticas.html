A linguagem da Web

O HTML é uma linguagem da web. É importante dizer que HTML não é uma linguagem de programação mas sim uma linguagem de marcação.

CSS é uma linguagem de estilização. É através da CSS que formatamos um texto, mudamos o tipo da letra fonte, sua cor, tamanho, colocamos bordas ou escondemos algum elemento.

É muito comum ouvir alguém dizer estou programando em HTML, isso está errado, o certo seria falar que você está escrevendo HTML ou codificando em HTML.
